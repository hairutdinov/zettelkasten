# Escape Sequences in Linux

| Sequence | Character printed        |
|----------|--------------------------|
| \a       | Alert or Ctrl-G (bell)   |
| \b       | Backspace or Ctrl-H      |
| \f       | Formfeed or Ctrl-L       |
| \n       | Return (Enter) or Ctrl-M |
| \t       | Tab or Ctrl-I            |
| \r       | Return (Enter) or Ctrl-M |
