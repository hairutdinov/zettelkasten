# Топология звезда
Компьютеры подключаются к центральному устройству. Передача данных от одного комп. к другому осуществляется через это устройство.
В качестве центр. устр. может служить:
- Коммутатор
- Концентратор
- Маршрутизатор
- Точка доступа Wi-Fi
